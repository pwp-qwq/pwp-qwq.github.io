欢迎使用 Windows Millennium Edition 操作系统！现在使用计算机会比以前更加轻松和方便。本教程将概括介绍 Windows Millennium Edition（即 Windows Me）。

在“新增功能”部分中，单击左面的主题可查看自 Microsoft Windows 上一版本后我们所做的一些改进，其中包括帮助您将计算机还原到以前状态或迅速修复故障的新工具，以及帮助无需关闭就可安全地离开计算机并可节省电能的解决方案等。

另外，单击本页上部的类别可了解其他 Windows Me 功能。例如，您还可以：

  • 在多台家庭计算机中创建一个网络
  • 前所未有地享受多媒体带来的冲击
  • 在计算机上自己制作电影

如果先需要一些基础知识，您可从“开始工作”开始。

如果在迟些时候要返回到本教程，只需单击“帮助和支持”中工具栏上的“教程和自学材料”即可。
