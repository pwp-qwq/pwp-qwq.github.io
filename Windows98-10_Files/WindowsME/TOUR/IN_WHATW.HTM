使用 Windows Millennium Edition 可在运行 Windows Me、Windows98 及 Windows 95 的计算机上简易安装家庭网络。按照以下三个基本步骤操作即可：

  1. 设置装有 Windows Me 操作系统的计算机。 “家庭网络向导”确保计算机作好网络方面的准备并按您所需的方式设置网络。如果您打算共享 Internet 连接，那么最好在此计算机上设置连接。
  2. 创建家庭网络安装盘。在“家庭网络向导”中创建此盘是非常简便的。
  3. 在每台计算机上进行安装。方法是：将“家庭网络安装”盘插入家庭网络中的每台计算机上，然后运行此向导。

如要更改“家庭网络”设置，请再次从“控制面板”中运行此向导，然后进行所需选项。

有关详情，请在“帮助和支持”的“搜索”框中键入“家庭网络”。
