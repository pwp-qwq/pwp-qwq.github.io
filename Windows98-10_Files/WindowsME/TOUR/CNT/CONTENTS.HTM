欢迎访问

Microsoft® Windows Media™ Player 是在计算机和 Internet 上播放和整理音频和视频文件的中心。使用 Windows Media Player，您可以：

  • 在 Internet 上查找音乐、视频内容和广播电台。
  • 播放音乐 CD 并将其复制到您的计算机上。
  • 根据您的喜好改变播放器的外观。
  • 创建您要播放的歌曲或视频节目的播放列表。
  • 将音乐复制到便携设备上。

立即播放

您可以使用“立即播放”功能观看视频节目或正在播放音乐的可视化效果。

可视化效果是随着音乐的节奏而闪耀的彩色几何形状和图案。

“立即播放”显示艺术家的姓名、音频曲目的标题和当前播放列表。

媒体指南

使用“媒体指南”，您可以直接在 Internet 上收听并观看音频和视频文件，或将它们下载到您的计算机上。“媒体指南”象一本电子杂志，它经常更新，提供到 Internet 上最新的电影、音乐和视频节目的链接。

CD
音频

在您的计算机上播放音乐 CD 将更加方便。当您将音频 CD 插入光盘驱动器之后，它自动开始播放。使用 CD 音频功能，可以实现：

  • 显示您正在倾听的 CD 的信息。
  • 从 Internet 上获取其它信息，例如评论和该艺术家的其它作品。
  • 制作 CD 副本供个人使用。
  • 将曲目从 CD 复制到您的硬盘上，然后按照您喜欢的顺序进行排列。

单击播放视频按钮观看该功能的演示。

媒体库

使用“媒体库”来整理您的媒体文件。您可以：

  • 保存指向计算机或 Internet 上文件的链接。
  • 创建您的计算机、网络或 Internet 上您喜欢的音频和视频内容的播放列表。
  • 自动将音频和视频内容按一定标准（例如唱片集标题、艺术家姓名、流派或作者）分成不同的类别。

单击播放视频按钮观看该功能的演示。

收音机调谐器

使用“收音机调谐器”，您可以找到世界各地的、几乎所有类型的广播电台。要找到某个地区的广播电台，您可以按照邮政编码或电台的呼号进行搜索。

如果您有多个喜欢收听的广播电台，您可以创建自己的预置频道，就象对汽车中的收音机所进行的操作一样。

单击播放视频按钮观看该功能的演示。

便携设备

您想随时随地聆听音乐吗？Windows Media Player 使您可以方便地将媒体文件传输到便携设备中，例如袖珍电脑和掌上电脑、Compact Flash 卡和其它设备上。

外观选择器

使用“外观选择器”功能，可以为 Windows Media Player 制作新的外观，享受其中的乐趣。应用外观就象是给播放器戴一副面具。您可以定制任何您喜欢的 Windows Media Player 外观来代替标准外观。Windows Media Player 附带 15 个不同的外观，在 Internet 上还可以获得其它类型的外观。

“外观选择器”功能显示可用外观的列表和外观预览。要更改外观，从列表中选择一个外观并单击“应用外观”，过程就这么简单。

单击播放视频按钮观看该功能的演示。
