非常感谢您对“Windows Update”站点的关注。


试用 Windows® Update 站点 - Windows 的 Web 扩展。使用“Windows Update”Web 站点可以 . . .
  • 请及时升级，以确保所使用的操作系统为最新版本。
  • 自动更新使用简便。
  • 随时查找您对 Windows 问题的答案。

要更好地利用“Windows Update”站点，您只需要使用 Internet Explorer 并与 Internet 连接。

连接到 Internet。
如果您从来没有和 Internet 连接过，请单击以上句子以启动 Internet 连接向导，此向导将帮助您登记服务。当您和 Internet 连接以后，请从“开始”菜单选择“Windows Update”图标，您就可以通过 Internet Explorer 直接进入“Windows Update”站点。如果您已经有 Internet 连接，请从“开始”菜单选择“Windows Update”图标。 Internet Explorer 将会启动“Windows Update”站点。

下次访问“Windows Update”站点。
单击以上的 X 图标以关闭浏览器窗口。执行如下操作，随时都可以创建到 Internet 的连接:

  • 转到“开始”菜单。
  • 选择“程序”组。选择 Internet Explorer，然后再单击“Internet 连接向导”。
  • Internet 连接向导将帮助您选择 Internet 服务提供商 (ISP) 或者将您连接到局域网 (LAN)。

连接到 Internet 后，请从“开始”菜单选择“Windows Update”站点，您就可以通过 Internet Explorer 进入 “Windows Update”站点。

我想从“开始”菜单删除“Windows Update”。
单击以上的 X 图标以关闭浏览器窗口。执行如下操作，随时都可以从“开始”菜单删除“Windows Update”图标:

  • 转到“开始”菜单。
  • 选择“Windows Update”站点。
  • 单击鼠标右键。
  • 单击“删除”。
